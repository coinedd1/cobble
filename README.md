#   cobble
A Kubernetes CLI tool for managing an Itzg Minecraft server.
## Features
* Pull crash and system logs seamlessly
* Various command shortcuts, no need to kubectl exec into rcon-cli
* More features in the works

# Installation 
### Download from GitHub
1. Navigate to the releases page
2. Download the "cobble" binary
3. If you want to use the CLI from anywhere (which you probably do), add the binary directory to your PATH, or point Go's binary directory to a directory already in your PATH.

### Compile with Go

1. Clone the repository

`git clone https://github.com/coinedd1/cobble.git`

2. Open the repository directory

`cd cobble` 

3. Build with the Go compiler. Make sure you have Go installed in your system.

`go build` or `go install` if you want a binary. 

4. If you want to use the CLI from anywhere (which you probably do), add the binary directory to your PATH, or point Go's binary directory to a directory already in your PATH.
