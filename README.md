#   cobble
A Kubernetes CLI tool for managing the itzg minecraft server image.
## Features:
* Pull crash and system logs seamlessly
* Various command shortcuts, no need to kubectl exec into rcon-cli
* Seamlessly swap between worlds

# Installation
### Compile with Go

Clone the repository:

`git clone https://github.com/coinedd1/cobble.git`

Open the repository directory

`cd cobble` 

Build with the Go compiler. Make sure you have Go installed in your system.

`go build` or `go install` if you want a binary. 

If you want to use the CLI from anywhere, add the binary directory to your PATH, or point Go's binary directory to a directory already in your PATH.
